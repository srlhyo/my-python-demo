GIT
========

// clone project into your local machine
git clone https://github.com/IamFred14/my-python-demo.git

// common git commands

git status // 

git add <file>  or git add . // to add all at once

git commit -m "dadsasda"  // message should reflect your changes

git push origin <branch>

git log // Shows all commits

git pull // Make sure current branch is updated

git branch -d localBranchName // delete branch locally

git push origin --delete remoteBranchName // delete branch remotely
